### Environmental Cost Worksheet Implementation

- Renamed from "Environmental Cost Worksheet" to "Cost Worksheet"
- Added lookup to site → valuation site → environmental site task group
- 28 expense fields now included in worksheet structure
- Budget vs. actual tracking for each expense category
  - ESA, PCA, TSA, RSRA as main products in phase 2
  - Additional less-used products to be mapped

### Cost Flow & Data Integration Process

- Order Service payload creates opportunity, site, environmental site, task group, and worksheets
- Travis provides cost estimate arrays → Mac builds worksheets → attaches to Salesforce payload
- Default values strategy:
  - Most estimates equal actuals initially (FOIA: $33, Historical: $185, etc.)
  - "Everything else" products: 50% of sale price goes to PA cost
  - Only PA cost field populated for miscellaneous items
- Professional Associate costs roll up to cost worksheet
- Internal expense object tracks travel, mileage, hotels, rental cars, plane tickets
