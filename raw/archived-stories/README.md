# Archived Stories

Stories that story-sync has pruned from `Action-Tracker.md` and per-project journals because they're no longer active (moved out of Todo/Unstarted/Started/In Progress and no longer meet backlog retention criteria).

## Layout

Stories archived by project:

- `{project-slug}/` — one folder per project
- `{project-slug}/{workspace}-{ticket-id}.md` — one file per archived story

## Fields preserved

- Full story description
- Final status
- Archived-on date
- Last-updated date before archive
- Project + workspace
- Link to original ticket

## Restoration

If a ticket becomes active again, story-sync automatically restores it from here on the next run.
